Projects

cuda_raytrace (repository)
The project originally started as a ray-traced image renderer that would process a scene with shadows and recursive reflections, but as the scene became more complex, the time required to generate the image grew exponentially. To combat the increasing time, the renderer was re-implemented using CUDA to utilize GPU-acceleration. The efficiency boost was significant enough to be viable in a real-time engine. The engine utilizes OpenGL's interoperability with CUDA in order to update a ray-traced scene in real-time.
